# lib

## Purpose
Shared frontend code accessed via `$lib/...`. Holds reusable Svelte components, rune-based stores, autogenerated Wails bindings, and the Material Icons font file.

## Key Files
| File | Description |
|------|-------------|
| `index.ts` | Placeholder barrel — empty |
| `MaterialIcons-Regular.woff2` | Local Material Icons font, referenced by `app.css` `@font-face` |

## Subdirectories
| Directory | Purpose |
|-----------|---------|
| `components/` | Reusable Svelte 5 components (List, Modal, MuiIcon, ScreenToggler, etc.). See `components/AGENTS.md` |
| `stores/` | Rune-based factory stores: `BibleStore`, `songsStore`, `screenStore`. See `stores/AGENTS.md` |
| `bindings/` | Wails-generated TS bindings (`changeme/backend/models`, `changeme/dbhandler`). Autogenerated — do not edit |

## For AI Agents

### Working In This Directory
- `bindings/` is rewritten by `wails3 generate bindings -d ./frontend/src/lib/bindings`. Edits will be lost; change `dbHandler.go` and regenerate via `task common:generate:bindings` (or the `wails3 dev` watch loop).
- Import path convention: `import { Foo } from "$lib/bindings/changeme/dbhandler"` for methods, `import { Book } from "$lib/bindings/changeme/backend/models"` for types.
- Font is bundled (`MaterialIcons-Regular.woff2`) — no network dependency. The `MuiIcon.ts` file is a hardcoded string-literal union of all valid icon names for type safety.

## Dependencies

### External
- `@wailsio/runtime` (Events, Screens namespaces).
