# src

## Purpose
SvelteKit source tree. Contains the three-tab operator UI, shared component library, reactive stores, and autogenerated Wails bindings.

## Key Files
| File | Description |
|------|-------------|
| `app.html` | SvelteKit HTML shell (default) |
| `app.css` | Tailwind layers + `@font-face` for Material Icons (woff2 in `lib/`) |
| `app.d.ts` | SvelteKit ambient types |

## Subdirectories
| Directory | Purpose |
|-----------|---------|
| `routes/` | SvelteKit routes — single `/` page with three tabs. See `routes/AGENTS.md` |
| `lib/` | Components, stores, generated bindings, font asset. See `lib/AGENTS.md` |

## For AI Agents

### Working In This Directory
- Single-route app: `+layout.svelte` only renders children + imports `app.css`; `+layout.ts` disables SSR and forces prerender.
- `+page.svelte` holds the navbar + tab switching. The three tab views (`Bible.svelte`, `Songs.svelte`, `Screens.svelte`) sit in `routes/` but are imported as plain components, not as nested routes.

### Common Patterns
- All UI text is Russian (Cyrillic). Keep it that way when adding labels — see "Библия"/"Песни"/"Экраны", "СКРЫТЬ"/"ПОКАЗАТЬ", etc.
- Material Icons via `<MuiIcon name="...">` — font is bundled locally, no Google CDN fetch.

## Dependencies

### Internal
- `lib/bindings/changeme/dbhandler` for backend calls; `@wailsio/runtime` for events + screens.
